# Chat TCP Unity

TCP Client-Server Model using .NET and Unity
Project written based in the NetCoreServer Library
NetCoreServer: https://github.com/chronoxor/NetCoreServer

## Unity (Client)
* **Dependency**: NetCoreServer.dll
* **Difficulty**: Intermediate
* **Version**: 2019.2.7f2
* **.NET version**: .NET 4.x
